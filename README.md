# Azaym House Boat Website

A promotional website for **Azaym House Boat**, designed to showcase a premium and memorable sea experience for customers seeking relaxation, celebrations, and unique private events.

## Overview

This project presents a modern and responsive website for a house boat rental service, allowing visitors to explore the boat, browse available packages, view pricing, and contact the business directly.

The website is designed with a clean user interface and smooth navigation to provide an enjoyable browsing experience across both desktop and mobile devices. It serves as a digital showcase for the business while making booking and communication simple and accessible.

## Features

* House boat image gallery
* Event and celebration packages
* Pricing section
* Direct booking via WhatsApp
* Social media integration
* Google Maps location
* Mobile-friendly design
* Swipe-enabled image sliders
* Smooth section navigation without page reload

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript

## Project Structure

* `index.html` for the website structure and content
* `style.css` for styling and layout
* `app.js` for interactive functionality
* `images2/` for storing all project images and media assets

## Purpose

The main purpose of this project is to provide an attractive online presence for Azaym House Boat, helping customers discover available services, explore event packages, and easily reach the business for booking and inquiries.
